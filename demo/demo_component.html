Main title: Building Static HTML Pages with Reusable JavaScript Components

Xây dựng trang HTML tĩnh với các component tái sử dụng bằng JavaScript

Introduction

Các website hiện đại không phải lúc nào cũng cần đến framework phức tạp. Bằng cách kết hợp HTML tĩnh với các component JavaScript có thể tái sử dụng, bạn có thể xây dựng những trang web nhẹ, dễ bảo trì và có khả năng mở rộng tốt. Cách tiếp cận này giúp tổ chức giao diện thành các phần nhỏ như card, code block hay section nội dung, từ đó làm code rõ ràng và dễ phát triển hơn. Đặc biệt phù hợp với trang tài liệu, blog cá nhân hoặc knowledge base. Trong bài viết này, chúng ta sẽ tìm hiểu cách xây dựng và tái sử dụng component bằng JavaScript thuần.

Hãy add file initComponents.js vào trong project: 

<script type="module" src="../../utils/components/initComponents.js"></script>
Demo

Sau đây là các ví dụ về cách sử dụng các component đã được định nghĩa trong file initComponents.js. Bạn có thể copy và paste trực tiếp vào trong file HTML của mình để sử dụng.

2.1 Section Header

Section Header: được sử dụng để hiển thị tiêu đề cho từng section nội dung, giúp phân chia rõ ràng các phần trong trang. 

<section-header number="0"> This is a section header </section-header>
This is a section header

2.2 Text Icon Item

TextIconItem: được sử dụng để hiển thị một dòng văn bản kèm theo icon.

<text-icon-item icon="→" iconColor="text-blue-600">
  <strong>Ví dụ:</strong>  Minh họa sử dụng  <code>text-icon-item</code> gồm icon mũi tên + text nằm trên 1 dòng.
</text-icon-item>
Ví dụ: Minh họa sử dụng text-icon-item gồm icon mũi tên + text nằm trên 1 dòng.

2.3 Alert Box

AlertBox: được sử dụng để hiển thị một thông báo quan trọng.

<alert-box title="Chú ý">
  bạn có thể chạy trực tiếp trang web này trên các host như Github Pages mà không cần backend phức tạp. Chỉ cần
  đảm bảo rằng các file HTML, CSS và JavaScript được tổ chức hợp lý trong repository của bạn.
</alert-box>
bạn có thể chạy trực tiếp trang web này trên các host như Github Pages mà không cần backend phức tạp. Chỉ cần đảm bảo rằng các file HTML, CSS và JavaScript được tổ chức hợp lý trong repository của bạn.

2.4 Card Info

CardInfo: được sử dụng để hiển thị thông tin dưới dạng card.

<card-info
  title="CardInfo title"
  items='[
    {
      "icon": "→",
      "iconColor": "text-blue-600",
      "content": "<strong> Step 1:</strong>  mô tả bước khởi tạo"
    },
    {
      "icon": "→",
      "iconColor": "text-blue-600",
      "content": "<strong> Step 2:</strong> mô tả bước sử dụng"
    }
  ]'
>
</card-info>
