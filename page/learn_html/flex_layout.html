Flexbox vs Grid: Khi nào dùng cái nào?

Trong CSS và Tailwind CSS, mục đích thiết kế của chúng hoàn toàn khác nhau.

Flex

Bố trí theo 1 chiều (hàng hoặc cột)

Grid

Bố trí theo 2 chiều (hàng và cột cùng lúc)

1. Flexbox là gì?

Flexbox phù hợp khi:

🎯 Căn giữa
↔️ Sắp xếp theo hàng/cột
📱 Navbar
🔘 Button group
📄 Card list đơn giản
✨ Layout theo nội dung
<!-- Ví dụ Tailwind --> <div class="flex gap-4"> <div class="bg-blue-200 p-4">Item 1</div> <div class="bg-blue-200 p-4">Item 2</div> <div class="bg-blue-200 p-4">Item 3</div> </div>
[ Item1 ] [ Item2 ] [ Item3 ]

2. Grid là gì?

Grid phù hợp khi:

📊 Dashboard
🖼️ Gallery
🌐 Layout web
📏 Chia cột rõ ràng
🗂️ Kiểm soát hàng + cột
<!-- Ví dụ Tailwind --> <div class="grid grid-cols-3 gap-4"> <div class="bg-green-200 p-4">1</div> <div class="bg-green-200 p-4">2</div> <div class="bg-green-200 p-4">3</div> <div class="bg-green-200 p-4">4</div> </div>
[ 1 ] [ 2 ] [ 3 ]
[ 4 ]

💡 Tư duy Hệ thống Lưới (12-Column Grid)

Khái niệm: Hãy tưởng tượng chiều rộng màn hình được chia thành 12 phần bằng nhau. Khi ta nói một phần tử chiếm col-span-4, nghĩa là nó đang lấy đi 4/12 (tương đương 1/3 hay 33.3%) diện tích hàng ngang.

Cấu trúc TRƯỚC (3/12)

<div class="grid grid-cols-12"> <div class="lg:col-span-3"> Nhật ký </div> <div class="lg:col-span-9"> Bản đồ </div> </div>

Cấu trúc SAU (4/12)

<div class="grid grid-cols-12"> <div class="lg:col-span-4"> Nhật ký </div> <div class="lg:col-span-8"> Bản đồ </div> </div>

⚓ Cấu trúc "Sticky Footer"

<!-- Thẻ bao ngoài cùng --> <div class="flex flex-col min-h-screen"> <header>...</header> <!-- Main sẽ tự "nở" ra để chiếm chỗ trống --> <main class="flex-grow"> Nội dung Grid nằm ở đây </main> <footer>...</footer> </div>

7. Tailwind class phổ biến

Flex
flex Bật flexbox
flex-col Chuyển thành cột
justify-center Căn ngang
items-center Căn dọc
gap-4 Khoảng cách
flex-wrap Xuống dòng
Grid
grid Bật grid
grid-cols-3 3 cột
grid-rows-2 2 hàng
col-span-2 Chiếm 2 cột
row-span-2 Chiếm 2 hàng
gap-4 Khoảng cách

Ví dụ hiện đại phổ biến

Responsive card layout:

<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6"> <!-- Nội dung các Card --> </div>

Bên trong mỗi Card (Flexbox):

<div class="flex items-center gap-3"> <img src="..." /> <span>Avatar</span> </div>
